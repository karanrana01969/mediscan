# Mediscan

Mediscan is a mobile application and FastAPI backend designed to help the elderly easily identify the use of medications in hand, ensuring they know what each medication is for, its side effects, and typical dosage.

## Project Structure

- `backend/` - Contains the Python FastAPI backend, database models, and Gemini AI integration.
- `mobile/` - Contains the React Native frontend application.

See `backend/README.md` for full API documentation.
